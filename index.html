Politique de confidentialité
5 Rois — Cinq Couronnes

Dernière mise à jour : 14 avril 2026

1. Collecte de données

Pour le mode multijoueur en ligne, l'application utilise Firebase (Google) pour gérer les salles de jeu en temps réel. Les données suivantes peuvent être temporairement stockées sur les serveurs Firebase :

  • Le pseudonyme choisi pour la partie
  • Le code de salle et les scores de la partie en cours

Ces données sont supprimées automatiquement à la fin de chaque partie. Aucune donnée n'est conservée de manière permanente sur nos serveurs.

2. Données stockées localement

L'application peut stocker localement sur votre appareil vos préférences (langue, pseudo) pour améliorer votre expérience. Ces données ne quittent jamais votre appareil.

3. Services tiers — Firebase

Le mode multijoueur utilise Firebase Realtime Database (Google LLC). En jouant en ligne, vous êtes soumis à la politique de confidentialité de Google : policies.google.com/privacy

4. Publicité

L'application ne contient aucune publicité.

5. Enfants

Cette application est adaptée à tous les âges. Aucune donnée personnelle identifiable n'est collectée.

6. Contact

Pour toute question concernant cette politique : verby188@github.com

7. Modifications

Cette politique peut être mise à jour. Les modifications seront publiées sur cette page.
